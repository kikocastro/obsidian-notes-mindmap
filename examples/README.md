# Examples

## `mindmap-demo`

A tiny self-contained map: 2 goals -> 3 projects -> 5 tasks, wired by `goal:` and `project:` frontmatter links. It exercises levels, edges, a status filter, and a progress bar.

To try it, copy the `mindmap-demo` folder into the **root** of your vault (the `from:` paths in `Mindmap demo.md` are vault-root-relative, e.g. `mindmap-demo/goals`), then open `Mindmap demo.md` in Reading view or Live Preview. If you put the folder elsewhere, repoint the three `from:` paths to match.
